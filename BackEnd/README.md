# ForeSee_BackEnd_SpringCloud

## 项目介绍

### 组织结构

```lua
ForeSee
├── client-121 -- 接收从前端发送过来的请求
├── eureka-222 -- 服务注册中心
├── server-192 -- 微服务提供者
├── gateway-222 --路由转发
```
